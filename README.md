# libyuarel

Simple C library for parsing URLs with zero-copy and no mallocs.

## To build

```sh
$ make && sudo make install
```

## Try it

Compile the example in `examples/`:

```sh
$ make examples
```

Run the example program:

```sh
$ ./simple
```

## How to use it:

See example in `examples/simple.c`. Compile with `-lyuarel`.
